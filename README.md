# Activity Service

RESTful API for interacting with activities. Lives at [http://sika-activity-api.herokuapp.com](http://sika-activity-api.herokuapp.com).

## Dependencies

* Docker / Docker Compose

## Stack

* Express
* Mongo
* Cucumber, Expect
* Supertest

## Scripts

* `./test` - Run tests
* `./dev` - Start dev environment (base URL is `localhost://3000`)
